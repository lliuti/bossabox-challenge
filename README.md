# VUTTR
## Very Useful Tools To Remember


#### Install all dependencies
In order to be able to run the API you must install all of it's dependencies. So, at the root folder you can run:

```
npm install
```
or
```
yarn
```


#### How to run it
Also at the root folder, execute:

```
npm run dev
```
or
```
yarn dev
```


#### Built with
* NodeJS
* Express
* Sequelize
* Nodemon
* Sucrase
